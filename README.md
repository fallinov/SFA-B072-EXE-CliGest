# Exercice GestCli

Support de cours : https://divtec.gitbook.io/quasar/


## Installer les librairies
```bash
yarn
# ou
npm install
```

### Démarrer l'application
```bash
quasar dev
```

### Inspecter les erreurs de qualité du code : Lint
```bash
yarn lint
# ou
npm run lint
```

### Créer la version de production
```bash
quasar build
```
